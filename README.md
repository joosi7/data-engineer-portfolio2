# Sales Data Pipeline

A Python ETL pipeline that processes raw sales data from multiple stores.

## What it does
- Reads raw CSV files from 3 different stores
- Validates and cleans every row
- Merges all stores into one clean dataset
- Analyses revenue by store and by product
- Saves a clean CSV and a professional report

## Skills demonstrated
Python · CSV processing · Data validation · ETL pipeline · Error handling

## How to run
python pipeline.py
